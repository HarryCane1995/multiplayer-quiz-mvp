# Multiplayer Quiz MVP

Локальное MVP мультиплеерного квиза на Next.js, TypeScript и Socket.io. Состояние комнат хранится в памяти сервера, без регистрации и базы данных.

## Запуск

Требуется Node.js 20+ и npm.

```bash
npm install
npm run dev
```

После старта откройте [http://localhost:3000](http://localhost:3000).

Для тестовой партии:

1. В первой вкладке нажмите `Создать игру`.
2. Во второй вкладке нажмите `Присоединиться`.
3. Введите код комнаты и никнейм.
4. На экране хоста нажмите `Начать игру`.
5. Игрок отвечает на вопросы, хост переводит игру дальше после завершения раунда.

## Структура файлов

- `server.ts` - кастомный Next.js сервер с Socket.io, in-memory комнатами, таймером, ответами и начислением очков.
- `src/lib/questions.ts` - пять тестовых вопросов с `text`, `options[]`, `correctIndex`.
- `src/lib/types.ts` - общие TypeScript-типы для сервера и клиентского UI.
- `src/lib/socket.ts` - singleton Socket.io-клиент для браузерных компонентов.
- `src/app/page.tsx` - главная страница с кнопками создания и входа.
- `src/app/host/page.tsx` и `src/components/HostGame.tsx` - отдельный экран хоста.
- `src/app/join/page.tsx` и `src/components/PlayerGame.tsx` - отдельный экран игрока.
- `src/components/Timer.tsx` - клиентский таймер вопроса.
- `src/components/ScoreTable.tsx` - таблица очков и результатов.
- `src/app/globals.css` - общие стили интерфейса.

## Правила начисления

- Первый правильный ответ в раунде получает `1000` очков.
- Остальные правильные ответы получают `500` очков.
- Неправильные ответы получают `0` очков.
- После ответа всех активных игроков или истечения таймера хост может перейти к следующему вопросу.

Состояние сбрасывается при перезапуске сервера.
